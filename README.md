# Targ Site

Современная платформа объявлений с акцентом на UX, мобильность и визуальную эстетику.

## Основные возможности

- Просмотр объявлений с детальной страницей в стиле glassmorphism и фирменных цветах
- Галерея изображений с полноэкранным просмотром
- Адаптивная верстка для мобильных и десктопных устройств
- Sticky top bar с кнопками "Назад" и "Поделиться"
- Кнопка "В избранное" с минималистичным глассморфичным стилем
- Контакт с продавцом и система отзывов
- Модальные окна для жалоб, отзывов, копирования ссылки и др.
- Использование React 19, TypeScript, TailwindCSS

## Структура проекта

- `src/components/listing/ListingPage.tsx` — основная страница объявления
- `src/hooks/` — хуки для адаптивности, локализации, изображений
- `src/contexts/` — контексты для авторизации, отзывов, диалогов
- `src/components/ReviewModal.tsx`, `SellerReviewsModal.tsx`, `ReviewRestrictionModal.tsx` — модальные окна
- `public/`, `build/` — статика и сборка

## Быстрый старт

1. Установите зависимости:
   ```sh
   npm install
   ```
2. Запустите проект:
   ```sh
   npm start
   ```
3. Откройте [http://localhost:3000](http://localhost:3000)

## Кастомизация и стили

- Все ключевые элементы используют TailwindCSS и кастомные классы для фирменного стиля
- Glassmorphism, плавные градиенты, современные кнопки, адаптивные контейнеры
- Для изменения цветовой схемы — редактируйте `tailwind.config.js`

## Особенности UI/UX

- Верхняя панель всегда под навигацией, с увеличенной высотой и закруглением
- Фото объявления — квадратное, крупное, без рамки, с плавным увеличением при наведении
- Блоки описания и характеристик — в одну строку на десктопе
- Кнопка "В избранное" — минималистичная, с прозрачным фоном и плавным выделением
- Все модальные окна стилизованы в едином стиле

## Технологии

- React 19
- TypeScript
- TailwindCSS
- Heroicons

## Локализация
